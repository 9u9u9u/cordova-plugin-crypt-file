# Cordova crypt file plugin
HTML source file is encrypted at build, and decrypted at run.  


## Encrypt
`cordova build [ios / android]`


## Decrypt
`cordova emulate [ios / android]`  
or  
`cordova run [ios / android]`  


##Encryption subjects.
* .html
* .htm
* .js
* .css


##Supported platforms
* iOS
* Android
